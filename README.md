# 📊 CodeAlpha Python Internship — Task 2: Stock Portfolio Tracker

Console tool that lets a user enter positions and computes total value using **hardcoded prices**.

## ✅ Scope (per CodeAlpha)
- User inputs **stock symbol + quantity**
- Use a **dictionary** of predefined prices
- Display total value and optionally **save to CSV**

## ▶️ Run
```bash
python portfolio_tracker.py
```

## 🧠 Concepts
dictionaries, input/output, arithmetic, optional file handling (CSV)

## 🎥 Demo Script (1–2 min)
- Show allowed symbols and add a few positions
- Display summary and total
- Choose "y" to save CSV and show the generated file
